Welcome Teachers

Welcome to Goal Mine, an easy-to-use data collection and reporting tool

Goal Mine was created by a special education teacher, for special educations teachers. It takes IEP goal tracking beyond binders and spreadsheets and puts it into a web application that you can use on your smart phone, tablet, or desktop computer.

Features

  • Template wizard for data collection
  • Customizable data collection methods for each item being tracked
  • Automatic date and time stamp for easy retrieval and report processing
  • Optional stopwatch function for timing-specific goals
  • Important event "markers" that allow you to document and view events such as medication changes along side goal progress.
