Welcome Teachers

Welcome to SPED Track, an easy-to-use data collection and reporting tool

SPED Track was created by a teacher, for teachers and takes IEP goal tracking out of the white binder and spreadsheets and puts it into a web application that you can use on your smart phone, tablet, or desktop computer.

Features

  • Template wizard for data collection
  • Automatic date and time stamp for easy retrieval and report processing
  • Optional stopwatch and timer functions for timing-specific goals
  • Important event "markers" that allow you to document and view events such as medication changes along side goal progress.
